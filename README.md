# 10ExerciciosPraticosJS
10 exercícios práticos de JavaScript para a matéria de PW(Programação Web)
1°: Verifica através da idade de uma pessoa(numeral) se ela pode dirigir.
2°: Recebe três números, e devolve qual deles é o maior.
3°: Recebe um valor numérico de uma moeda ou não, e diz qual moeda seria, ou se não seria uma moeda.
4°: Usuário cria uma senha, e o código verifica se é forte, a partir de específicos quesitos.
5°: Programa recebe uma temperatura em Graus Celsius e devolve como está o ar.
6°: Recebe um número de 1 a 7 e mostra um dia da semana, de forma respectiva com a ordem crescente de um a sete.
7°: Verifica os lados de um triângulo, e retorna o tipo dele.
8°: Usa concatenação para unir dois nomes e criar um nome completo.
9°: Recebe duas notas, faz a média delas e diz se o usuário está aprovado ou não, a partir do mínimo 7 para ser aprovado.
10°: Recebe um número de telefone, e o transforma no formato de um número telefônico funcional.
